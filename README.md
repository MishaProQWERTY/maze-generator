# Генератор лабиринтов

Интерактивное веб-приложение для генерации и визуализации лабиринтов с использованием алгоритма Вильсона (Wilson's algorithm).

## Возможности

- **Генерация лабиринтов** с настраиваемыми параметрами
- **Поиск пути** от входа до выхода алгоритмом A*
- **Визуализация решения** с отметкой пути красным цветом
- **Сохранение картинки** лабиринта в формате PNG
- **Адаптивный интерфейс** для мобильных устройств и десктопов
